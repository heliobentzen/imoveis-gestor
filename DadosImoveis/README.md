# Configuração do JSON Server
 Este documento fornece instruções para configurar o JSON Server em seu projeto.

## 1. Instalação
Para instalar o JSON Server, você precisa ter o Node.js e o npm instalados. Execute o seguinte comando no terminal:
npm install -g json-server

## 2. Execute o JSON Server apontando para o arquivo:
json-server --watch db.json --port 4000
